  About branches
1) git branch   → shows branch name
2) git branch -a   → list of all branches
3) git branch <branch-name>   → creates
4) git branch -d <branch-name>   → deletes
5) git branch -r   → listing all remote branches
6) git checkout <branch-name>   → switches
7) git checkout -b <new-name>   → creates & switch
8) git remote prune origin   → branch deleted on remote but locally not
9) git fetch <remote-name> <branch-name>
     ↳ Fetching specific remote branch



1)git merge
    git checkout main         # Switch to the main branch
    git merge branch-name     # Merge branch-name into main

2)Rebasing a Local Branch onto a Remote Branch
    git fetch <remote_name>
	git rebase <remote_name>/<branch_name>
    Example:
	git fetch origin
	git rebase origin/main

3)Undoing Changes
   1)git reset<file> : Removes the specified file from the staging area but leaves the working directory unchanged 
   2)git revert <commit> : Unchange from commit 

4)View History:
    1)git log or git log --oneline  - Shows a history of commits in the repository
    2)git diff- Displays differences between various commits working directory, and the staging area
    3)git show <commit>: Shows the details of a specific commit

5)git restore filename - undo changes made to file before staging 
6)git commit --amend -m "Corrected commit message"- correct committed with the wrong message


7)Recovering deleted branches 
   	git reflog
	git checkout -b feature-ui <commit_hash>

8)download the latest changes from the remote without merging 
       	git fetch origin
8.1)Fetching a Specific Remote Branch
        git fetch <remote_name> <branch_name>

11)don’t want Git to track changes to .log files or node_modules
       Create or edit a .gitignore file: 
             In gitignore
                 *.log
                 node_modules/
             If Git was already tracking those files, you need to untrack them first:
                       git rm -r --cached node_modules
                       git rm --cached *.log
                       git commit -m "Stop tracking logs and node_modules"
