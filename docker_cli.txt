========================
 Docker CLI 
========================

Basic Commands:
---------------
docker --version
    -> Shows the installed Docker version.

docker pull hello-world
    -> Downloads the hello-world image from Docker Hub.

docker images
    -> Lists all Docker images available locally.

docker run hello-world
    -> Runs a container from the hello-world image.

docker ps -a
    -> Shows all containers (running + stopped).
docker ps
    -> Shows only running containers.

docker rm <container_id>
    -> Deletes a stopped container.

docker rmi <image_name>
    -> Removes a Docker image.


Working with Redis Example:
---------------------------
docker pull redis
    -> Pulls the official Redis image.

docker run --name myredis -d redis
    -> Runs a Redis container in detached mode with name "myredis".

docker ps
    -> Shows currently running containers.

docker exec -it myredis redis-cli
    -> Opens Redis CLI inside the container.
       Example commands inside Redis:
       SET name "Alice"
       GET name
       exit

docker stop <cid|name>
    -> Stops a running container.

docker rm <cid|name>
    -> Deletes the stopped container.

docker rmi <image_name>
    -> Deletes an image.

Python App Dockerfile Example:
------------------------------
FROM python:3.9
WORKDIR /app
COPY . /app
RUN pip install -r requirements.txt
EXPOSE 5000
CMD ["python", "app.py"]

docker build -t mypythonapp .
    -> Build the image.

docker run -p 5000:5000 mypythonapp
    -> Run the Python app on port 5000.
